# 任务拆分总览（由 PLAN.md 映射）

本目录将 `PLAN.md` 的 15 个章节按同编号拆分为独立任务文件，确保每个细节都有落地执行项。

## 文件映射
- `task-01.md`：项目定位与目标（含非目标）
- `task-02.md`：技术栈与版本基线
- `task-03.md`：系统架构与核心链路
- `task-04.md`：数据库设计与建模
- `task-05.md`：REST 接口设计与契约
- `task-06.md`：第三方 AI 接口抽象与路由
- `task-07.md`：推荐引擎规则设计
- `task-08.md`：前端实现设计（Vue3）
- `task-09.md`：安全与合规
- `task-10.md`：日志、监控与异常
- `task-11.md`：测试设计与验收
- `task-12.md`：部署方案
- `task-13.md`：里程碑与交付物
- `task-14.md`：风险与应对
- `task-15.md`：MVP 范围与优先级

## 执行顺序建议
1. `01 -> 02 -> 03 -> 04 -> 05 -> 06 -> 07`
2. `08` 与 `09/10/11` 可并行推进
3. 最后执行 `12 -> 13 -> 14 -> 15` 做收口
