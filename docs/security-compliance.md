# 安全与合规设计（Task 09）

## 1. 安全控制
1. JWT 鉴权：除 `/auth/login` 外接口均需认证。
2. 参数校验：请求 DTO 使用 Hibernate Validator。
3. SQL 防注入：使用参数化查询/ORM。
4. 文件上传限制：
   - 类型：jpg/jpeg/png/webp
   - 大小：不超过 5MB

## 2. 合规控制
1. 前端与接口文档统一提示：  
   “本系统仅提供健康管理建议，不构成医疗诊断。”
2. 隐私最小化：仅存储实现功能所需字段。
3. API Key 仅保存于服务端环境变量，不入库、不入日志。

## 3. 审计要求
1. 敏感数据脱敏：token、api key、账号标识掩码处理。
2. 关键操作记录 traceId，便于问题追踪。
